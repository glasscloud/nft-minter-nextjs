# NFT Minter

Project built for a buyer to mint an NFT on Ethereum. Bootstrapped with NextJS.

## Getting Started

1. Clone the repo
`git clone https://github.com/glasscloud/nft-minter-nextjs`
2. Install dependencies
`yarn` 
3. Start dev server
`cd <project-folder> && yarn dev`
